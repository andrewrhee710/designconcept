Grid Designs: An essential part of any website.

Let's figure out why grids are so important.

First, what is a grid-based layout?

A grid-based layout is one that follows the pattern of a grid to organize elements throughout a webpage. These elements can be text, dividers, images, and more.

Adobe writer and editor Nick Babich cites grids as being a long-used tool of visual artists. This is because grids are a network of intersecting lines, so if elements are situated according to the grid then it will be more visually pleasing and easier to comprehend.

Grids can also be "responsive", which is when they respond differently depending on the device they are being displayed on. Changing the size of the screen the webpage is housed on will change the layout of the elements if the grid is reponsive.

Ruark Jewell, Pexels.com, CC-0, 2018

Grids are found everywhere in good design, not just in webpages. Architecture and building design is one of the most common examples that utilizes grid structures to create safe and beautiful buildings (think of a skyscraper in NYC and how it follows the classic grid design). It's visually pleasing to the eye.

Now, why are they useful?

Grids give shape and structure to a website. This structure is useful in helping a user figure out how to navigate through a website. This navigation can be how the reader reads the content on your page, how easily they can click through different links, etc.

As mentioned above, responsive grids can be incredibly useful for making the website accesible on mulitple platforms and devices. If the grid is able to adjust and shrink based on the size it's being viewed on, it has a greater capacity to be viewed across different mediums.

Think of a newpaper like the New York Times. Both the physical paper and online website are set up using a hierarchical grid, one that orders elements from top to bottom based on their importance. This organization makes it easier for readers to seperate different stories and process which is the most important in a split second. This same grid layout is utilized across several websites for the fast comprehension it allows readers.
